如果一个数字是3的某次幂，那么这个数一定只含有3这个质数因子
1162261467是int型范围内，最大的3的幂，它是3的19次方
这个1162261467只含有3这个质数因子，如果n也是只含有3这个质数因子，那么
1162261467 % n == 0
反之如果 1162261467 % n != 0 说明n一定含有其他因子

下面的while循环就是为了找到在int型范围下的最大的3的幂
只对质数有效

循环，如果INT_MAX不除3，那么会溢出；最后当p == INT_MAX / 3 后，会最后一次执行循环，再乘上一个3，此时就为该类型范围下的最大的3的幂
