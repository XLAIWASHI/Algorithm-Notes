## 容器方法
用容器是很好做的，我们可以用哈希表，把一个链表的所有节点地址存入(一定要是地址，只有地址才能判断是否是相同节点)；之后再遍历第二个链表，判断是否在哈希表中
### 代码
```cpp
class Solution {
public:
    ListNode *getIntersectionNode(ListNode *headA, ListNode *headB) {
        if(headA == nullptr || headB == nullptr)
        {
            return nullptr;
        }
        unordered_set<ListNode*> st;
        ListNode* cur = headA;
        while(cur)
        {
            st.insert(cur);
            cur = cur->next;
        }
        cur = headB;
        while(cur)
        {
            if(st.find(cur) != st.end())
            {
                return cur;
            }
            cur = cur->next;
        }
        return nullptr;
    }
};
```
自己看吧，这个还是好懂的

## 非容器方法
非容器方法，我们可以先遍历一下长的链表；判断和短的直接的差值是多少，在让长的提前走对应的长度，之后两个链表同步走，第一个相同的节点就是相交节点；

还有一个技巧：
	如果两个节点的最后节点是一个，那么这两个链表一定相交
### 代码
```cpp
class Solution {
public:
    ListNode *getIntersectionNode(ListNode *headA, ListNode *headB) {
        if(headA == nullptr || headB == nullptr)
        {
            return nullptr;
        }
        int diff = 0;
        ListNode* a = headA;
        ListNode* b = headB;
        while(a)
        {
            a = a->next;
            diff++;
        }
        while(b)
        {
            b = b->next;
            diff--;
        }
        if(a != b)// 如果最后一个不是同一个，那么一定不相交
        {
            return nullptr;
        }
        if(diff > 0)// a是长的链表
        {
            a = headA;
            b = headB;
        }
        else
        {
            a = headB;
            b = headA;
        }
        diff = abs(diff);
        while(diff--)// 让长的链表先走
        {
            a = a->next;
        }
        while(a != b)
        {
            a = a->next;
            b = b->next;
        }
        return a;
    }
};
```
