## 介绍
就是字面意思，只用递归，逆序一个栈

## 代码
```cpp
// 用递归逆序栈
#include <iostream>
#include <stack>

void reverse(std::stack<int>& sk);
int bottomOut(std::stack<int>& sk);

int main(void)
{
    std::stack<int> sk;
    sk.push(3);
    sk.push(2);
    sk.push(1);

    reverse(sk);
    while(!sk.empty())
    {
        std::cout << sk.top() << std::endl;
        sk.pop();
    }

    return 0;
}

void reverse(std::stack<int>& sk)
{
    if(sk.empty())
    {
        return;
    }
    int num = bottomOut(sk);
    reverse(sk);
    sk.push(num);
}

// 取出栈底元素，上面的元素盖下来
int bottomOut(std::stack<int>& sk)
{
    int ans = sk.top();
    sk.pop();
    if(sk.empty())
    {
        return ans;
    }
    int last = bottomOut(sk);
    sk.push(ans);
    return last;
}
```

## 讲解
![image-1782315294152.png](../../assets/05%20用递归的方式逆序栈/image-1782315294152.png)
这是bottomOut函数，他的作用就是，把栈底的元素拿出来，上面的元素盖下去
**流程：** 
1. 每次下一层，先把当前层的值弹出，再去递归，拿下面的值
2. 直到栈为空，此时，当前层的值就是栈底，我们返回
3. 上层接收到栈底元素后，把自己的值压入栈中，再次向上层返回栈底元素
这样，栈底元素就被拿到了，同时，上面的元素也盖下去了

![image-1782316004247.png](../../assets/05%20用递归的方式逆序栈/image-1782316004247.png)
这是reverse函数
**流程：** 
1. 每次通过bottomOut函数，拿到栈底元素，并更新栈情况
2. 直到栈为空，此时，返回
3. 我们把当前层的num压入栈中，因为，num是当时的栈底元素，所以最后反转成功
把每一层的栈底元素保存起来，等递归结束时，再把之前保存的栈底元素压回栈顶
